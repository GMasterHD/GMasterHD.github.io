Was ist der 1.17 SMP Server?

Der 1.17 SMP Server ist wie ein ganz normaler Survival Server
mit dem Unterscheied, dass dieser einige Plugins verwendet um das
Survival Erlebnis ein bisschen zu erweitern ohne von Vanilla zu weit
weg zu kommen.

Zu diesen Plugins zählen z.B. einfache Plugins wie AFKPlus, welches anzeigt
ob ein Spieler gerade AFK ist wiederum auch komplexere Plugins wie Dynmap
welches für die Live Map verantwortlich ist.

Um auf dem 1.17 SMP Server spielen zu können, müsst ihr von Leuten die bereits
auf dem Server spielen eingeladen werden.
Eingeladen Spieler werden überprüft und wenn man ihnen vertraut werden sie zur
Whitelist hinzugefügt.

Abstimmungen

Um an Abstimmungen für den Server teilzunehmen,
schaut einfach auf meinem oder BlueRayHD's Discord vorbei!

Plugins

Wie oben schon erwähnt verwendet der Server Plugins um das Survival Felling
ein bisschen zu erweitern.
Zu diesen Plugins zählen eigenprogrammierte Plugins wie SMP+, SMPMusic und SMPDiscord
wiederum auch von der Spigot/Paper Community programmierte Plugins wie Dynmap und LuckPerms.
Damit der Server wiederum auch sicher vor Cheatern ist benutzt er ein AntiXRay Plugin.

Custom Crafting

Es gibt nicht viele aber ein paar Custom Crafting Rezepte.
Alle aktuellen Rezepte findet ihr auf meinem Discord!

Live Map

Damit man jeder immer erfahren kann was auf dem Server geschieht,
hat der Server eine Live Map instaliert. Diese zeigt nahezu in Echtzeit
wo sich Spieler befinden, gebaute Gebäude und erkundete Chunks.

Um die Livemap auf einem eigenem Tab zu öffnen klicke einfach auch den Titel
oder sieh sie dir unten einfach an.
